About the Project

Welcome to Shopaholic. This website is built as a part of my course work for CS 5610. This website serves as a one stop place for listing out all the information about a wide range of electronic products.

Project Website

Shopaholic was built using the MEAN(MongoDB, Express, Angularjs and Node.js) stack. This website allows users to view a wide range of electronic gadgets under one umbrella. The admin of this website can manage users for the application.
